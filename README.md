
# CasaDispensa

WebApp/PWA per gestire dispensa di casa e lista spesa.

## Pubblicazione GitHub Pages

1. Caricare tutti i file nel repository
2. Andare in Settings → Pages
3. Source: Deploy from branch
4. Branch: main

## Struttura

index.html
manifest.json
service-worker.js
css/style.css
js/app.js
icons/

## PWA

Installabile su smartphone e convertibile in APK tramite:
https://www.pwabuilder.com
